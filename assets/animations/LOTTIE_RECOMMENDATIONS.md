## 📥 How to Download

### Method 1: Direct Download
1. Visit https://lottiefiles.com
2. Search for the animation
3. Click on the animation you like
4. Click "Download" button
5. Select "Lottie JSON"
6. Save to `assets/animations/` folder
7. Rename to match the required name:
   - `welcome_player.json`
   - `welcome_owner.json`
   - `welcome_admin.json`

### Method 2: Use LottieFiles Plugin (Recommended)
1. Install LottieFiles plugin for your IDE
2. Browse animations directly in your editor
3. Download with one click
4. Automatically saves to correct location

## 🎨 Animation Criteria

When choosing animations, look for:

### ✅ Good Animations
- **Duration**: 2-3 seconds (not too long)
- **Style**: Clean, modern, professional
- **Colors**: Can be customized or neutral
- **Size**: Under 100KB (for performance)
- **Loop**: Can loop or play once
- **Complexity**: Not too detailed (performs better)

### ❌ Avoid
- Very long animations (>5 seconds)
- Too many colors (hard to match theme)
- Very large files (>500KB)
- Overly complex animations (performance issues)
- Animations with text (language issues)

## 📝 File Naming

**Important**: Files must be named exactly as shown:

```
assets/animations/
├── welcome_player.json   ← For players
├── welcome_owner.json    ← For field owners
└── welcome_admin.json    ← For admins
```

If files are named differently, the app won't find them and will use fallback icons.

## 💡 Pro Tips

1. **Keep it Simple**: Simple animations load faster and look cleaner
2. **Match Your Brand**: Choose animations that match your app's style
3. **Test on Device**: Animations may look different on device vs desktop
4. **Optimize Size**: Use LottieFiles optimizer to reduce file size
5. **Have Fallbacks**: App automatically uses icons if Lottie files missing
