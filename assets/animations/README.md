# Lottie Animations

Download free Lottie animations from [LottieFiles.com](https://lottiefiles.com)

## Required Animations:

1. **success_checkmark.json**
   - Search: "success checkmark" or "success animation"
   - Use for: Booking confirmation success state
   - Recommended: Green checkmark with celebration effect
   - Download link example: https://lottiefiles.com/animations/success

2. **empty_bookings.json** (Optional)
   - Search: "empty state" or "no data"
   - Use for: Empty bookings list
   - Recommended: Simple, sporty illustration

3. **loading_football.json** (Optional)
   - Search: "football loading" or "soccer ball"
   - Use for: Custom loading states
   - Recommended: Spinning football/soccer ball

## How to Download:

1. Go to https://lottiefiles.com
2. Search for the animation type
3. Click on the animation you like
4. Click "Download" → Select "Lottie JSON"
5. Save the file to this folder with the name above
6. The app will automatically use it

## Note:

If you don't add these files, the app will use fallback UI (icons/text) instead.
